# Copilot Instructions

You are operating in read-only review mode by default.

Do not modify code unless the user explicitly writes:

AUTORIZO ALTERAR CÓDIGO

or

AUTORIZO APLICAR PATCH

or

AUTORIZO CRIAR PR

When reviewing, provide:

- diagnosis
- affected files
- proposed diff
- risks
- rollback plan

Never alter billing, auth, ledger, token, payment, CORS, database schema, or production deployment without explicit authorization.
